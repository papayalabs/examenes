Examenes ee una nueva funcionalidad mediante la cual, el administrador de la misma pueda crear ex√°menes para ser completados por los usuarios.


Instalacion Local
------------------

  1. Instalar Rails

  2. Clonar Repositorio

    git clone https://github.com/papayalabs/examenes.git

  3. Configurar base de datos.

    edit config/database.yml

  4. Instalar gemas

    bundle

  5. Migrar base de datos

    rake db:migrate

  6. Migrar semilla

    rake db:seed

  7. Iniciar el Servidor

    rails s -u puma
